BaseColor + Alpha Packer

BaseColor画像のRGBと、Alpha画像のグレースケールを1枚のPNGにまとめます。
Unity Built-in の Standard Shader（Transparent）で使う前提です。処理はブラウザ内だけで完結します。

ここにBaseColor画像をドラッグ＆ドロップ / クリックして選択
BaseColor Preview
ここにAlpha画像をドラッグ＆ドロップ / クリックして選択
Alpha Preview

オプション

通常はチェック初期値のままでOKです。Alpha画像が白黒画像なら、そのまま使えます。

結果プレビュー

出力PNGは RGB = BaseColor、A = Alpha です。UnityではこのPNGを Albedo に入れてください。
